All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-18

### Added
- Initial release of PACKAGE.broker.
- Core proxy logic with caching (D1 + KV).
- Artifact storage in R2.
- UI Dashboard for managing repositories and tokens.
- Support for GitHub, GitLab, and Bitbucket authentication.
- Analytics Engine integration.
